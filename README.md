# WeConnect - 微信智能消息助手

本地运行的微信网页版自动化操作工具，支持最多 5 个消息来源的监控、自动分类和回复。

## 快速启动

```bash
# 1. 安装依赖
npm run setup

# 2. 开发模式运行（前后端同时启动）
npm run dev

# 3. 或者先构建再启动（生产模式）
npm run build && npm start
```

启动后访问 `http://localhost:5173`（开发模式）或 `http://localhost:3210`（生产模式）

## 功能说明

### Step 1 - 启动页
点击启动按钮，通过 Playwright 打开微信网页版（wx.qq.com），用手机扫码登录

### Step 2 - 消息设置
配置最多 5 个消息来源（群聊/个人），每个来源可设置：
- 自动回复开关 / 回复对象
- 回复语气（专业友好/正式严谨/轻松活泼/简洁高效）
- 回复态度（积极主动/稳健审慎/热情服务/中立客观）
- 分类关键词和回复模板
- MCP 外部服务集成

### Step 3 - 操作仪表盘
- **左侧 (4/5)**: 日期选择 + 5 个消息来源 Tab + 对话记录列表
- **右侧 (1/5)**: 待办事项面板（今日新增 + 历史待办）

## 技术栈

| 组件 | 技术 |
|------|------|
| 前端 | React 18 + Vite |
| 后端 | Node.js + Express |
| 数据库 | JSON 文件（data/ 目录） |
| 浏览器自动化 | Playwright (Chromium) |
| 实时通信 | WebSocket |

## 系统要求
- Node.js 18+
- Windows 10+ / macOS 12+
